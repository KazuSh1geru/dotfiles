# 24のモチベーション効果テーブル

ME Dictionary「6つのマジック／24のモチベーション効果」の自分への問いかけ形式。
全効果はフラット（重みなし）。状況に合うものを選ぶだけ。

## Unfreeze（解凍）: 相対化させる

### 時間のマジック — 時間軸を切り替えて揺らぎを与える

| 効果 | 問いかけ |
|---|---|
| ロングレンジ | 「3年後の自分から見て、このモヤモヤは何？」 |
| ショートレンジ | 「今日1日だけで考えたら、何をすべき？」 |
| タイムシフト | 「半年前の自分が同じ場面にいたら、どう感じた？」 |
| デッドライン | 「これをいつまでに解決する？期限は？」 |

### 空間のマジック — 視界を切り替えて揺らぎを与える

| 効果 | 問いかけ |
|---|---|
| ズームアウト | 「事業全体から見て、この問題はどこに位置する？」 |
| ズームイン | 「今やるべき1つだけに絞ると何？」 |
| アングル | 「上司/メンバー/顧客から見たら、これはどう映る？」 |
| ロールプレイング | 「広木さんなら/穂高さんなら、何と言う？」 |

## Change（変化）: 方向づける

### 目標のマジック — 変化の方向性の魅力を高める

| 効果 | 問いかけ |
|---|---|
| ラダー | 「なぜこの変化が重要？事業/キャリアとの接続は？」 |
| オプション | 「選択肢は何がある？自分で選んでいるか？」 |
| ライバル | 「この領域で一歩先にいる人は誰？何が違う？」 |
| ロールモデル | 「なりたい像は？その人ならどう動く？」 |

### 安心のマジック — 変化への不安を除去する

| 効果 | 問いかけ |
|---|---|
| マイルストーン | 「最初の1歩は何？明日できることは？」 |
| エスコート | 「誰に伴走してもらえる？相談先は？」 |
| マッサージ | 「同じ状況の人は？話せる相手は？」 |
| セーフティネット | 「失敗しても何が残る？最悪ケースは？」 |

## Refreeze（再凍結）: 固める

### 習慣のマジック — 新たな行動を習慣にする

| 効果 | 問いかけ |
|---|---|
| コミットメント | 「誰に何を宣言する？」 |
| ルーティン | 「いつ・どこで・何をルーティンにする？」 |
| モニタリング | 「何で進捗を測る？見える化は？」 |
| フィードバック | 「誰からいつフィードバックをもらう？」 |

### 集団のマジック — 集団の影響力で変化を固める

| 効果 | 問いかけ |
|---|---|
| スポットライト | 「この変化を誰の前で見せる？」 |
| サンクス | 「変化を支えてくれた人に感謝を伝える機会は？」 |
| ファーストクラス | 「変化した先にどんな特権がある？」 |
| ウェーブ | 「周囲で同じ変化をしている人は？一緒にやれないか？」 |
