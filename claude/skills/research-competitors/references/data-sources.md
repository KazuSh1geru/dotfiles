# 競合調査データソース

## 既存データ

### research/competitors/競合調査.csv

パフォーマンスマネジメント・人事評価領域の競合調査データ。

#### カラム構成

| カラム名 | 説明 | 例 |
|---------|------|-----|
| 企業名・サービス名 | サービス名 | カオナビ、Lattice |
| ITReview レビュー件数 | 日本のレビュー数（-は海外のみ） | 425、- |
| パフォーマンスマネジメント領域のサービス内容 | 機能説明 | 評価シートをクラウド化... |
| 価格（月額・1ユーザー・円換算） | 料金情報 | 要問い合わせ、$14 → 2212円 |
| 解決する経営課題 | ターゲット課題 | 属人的な評価業務の効率化 |
| 口コミの内容 | ユーザーの声の要約 | UIが直感的で使いやすい |
| サービス公式URL | 公式サイト | https://www.kaonavi.jp |
| レビューURL | レビューサイト | ITreview/G2のURL |

#### 主なカテゴリ

**日本市場（ITreview掲載）**
- カオナビ（425件）
- タレントパレット（376件）
- HRBrain（210件）
- あしたのクラウド（39件）
- その他多数

**グローバル市場（G2掲載）**
- Lattice（4046件）- 最大手
- Leapsome（2172件）
- HiBob（2418件）
- 15Five（1848件）
- Culture Amp（1527件）
- その他多数

#### 特徴

- 日本製品と海外製品の両方を網羅
- レビュー件数でユーザー規模を推測可能
- 価格情報あり（要問い合わせ多い）
- 口コミから強み・弱みを把握可能

## URL深掘り調査のポイント

### 公式サイトから取得すべき情報
- 最新の機能アップデート
- 料金プラン詳細
- 導入事例・ケーススタディ
- ターゲット顧客の明示
- 他社との差別化ポイント

### レビューサイトから取得すべき情報
- 星評価の分布
- Pros/Consの傾向
- どんな規模・業種の企業が使っているか
- 導入の決め手
- 乗り換え前のツール
- サポートの評価

## データ更新

新しい競合を発見した場合:
1. 競合調査.csvに行を追加
2. 同じカラム構成で情報を記入
3. URLは必ず記入（深掘り調査に必要）
